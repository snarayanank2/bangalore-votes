# Foundation & Ward Finder Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the production skeleton — Astro SSR app, Postgres, nginx micro-cache, bilingual routing — and ship the ward finder: home page with pincode + address lookup, and SEO-ready ward pages with boundary maps, from the real 369-ward dataset in `data/gba.geojson`.

**Architecture:** One Astro 5 SSR monolith (Node adapter) behind nginx (micro-cache, rate limiting) with Postgres, per `docs/superpowers/specs/2026-07-17-production-architecture-design.md`. English pages at root, Kannada under `/kn/`. Public HTML never varies by session. Ward polygons load into app memory for point-in-polygon; per-ward GeoJSON files are served statically for MapLibre.

**Tech Stack:** Node 22 · Astro 5 (`@astrojs/node` standalone) · TypeScript strict · Postgres 17 + Drizzle · nginx 1.27 · Turf.js · MapLibre GL · Vitest · Playwright

**Plan sequence** (this is Plan 1; each later plan gets its own document when its turn comes):
1. **Foundation & ward finder** ← this plan
2. Auth & registration (OTP, sessions, `/api/me` personalization, `/account/*`, Register/Login modal)
3. Curator tooling & candidate data (report cards, compare, translation pipeline)
4. Citizen contribution (flags, issue votes, curator queue)
5. Admin, partners & EOI
6. Editorial content & guides, full SEO/AEO polish
7. Comms & jobs (send calendar, readiness gating, backups), deploy/TLS hardening

## Global Constraints

- Node 22.x; TypeScript `strict: true`; Astro 5 with `output: 'server'` and `@astrojs/node` mode `standalone`.
- English at root paths, Kannada at `/kn/` prefix. Every public page emits `hreflang` alternates for `en`, `kn`, and `x-default`.
- **Public page HTML must never vary by session.** No `Set-Cookie` on public GET responses (nginx refuses to cache any response that sets one).
- **`/api/ward-lookup` responses must never contain coordinates** — return a ward, not a position (Google Maps ToS; `docs/project-dependencies.md` §6.4). Put this sentence as a comment in the geocoding module.
- All user-facing copy goes through `src/i18n/en.json` / `kn.json` — no hardcoded strings in templates. Kannada values are machine-draft; mark the file header for native-speaker review in PR.
- Zero client-side JavaScript by default. A `<script>` appears only where a task explicitly adds one.
- Conventional commit messages (`feat:`, `test:`, `chore:`). Commit at the end of every task.
- Ward IDs are the official ward numbers (1–369) from `data/gba.geojson` `properties.ward_id`.

- [ ] **Step 3: Create deploy/nginx.conf**

```nginx
worker_processes auto;

events { worker_connections 1024; }

http {
  proxy_cache_path /var/cache/nginx levels=1:2 keys_zone=pages:10m max_size=200m inactive=10m;
  limit_req_zone $binary_remote_addr zone=api:10m rate=10r/s;

  # Dynamic route prefixes are never cached (both languages).
  map $uri $skip_cache {
    default 0;
    ~^/(api|account|curator|admin|login) 1;
    ~^/kn/(account|curator|admin|login) 1;
  }

  upstream app {
    server app:4321;
  }

  server {
    listen 80;

    location /api/ {
      limit_req zone=api burst=20 nodelay;
      proxy_pass http://app;
      proxy_set_header Host $host;
      proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    location / {
      proxy_pass http://app;
      proxy_set_header Host $host;
      proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;

      proxy_cache pages;
      # One cached variant per URL: query params (e.g. ?src=partner) never
      # fragment the cache. Public HTML must not vary by session (spec §5).
      proxy_cache_key $host$uri;
      proxy_cache_methods GET HEAD;
      proxy_cache_valid 200 60s;
      proxy_cache_bypass $skip_cache;
      # Guard the invariant: any response that sets a cookie is not cached.
      proxy_no_cache $skip_cache $upstream_http_set_cookie;
      add_header X-Cache-Status $upstream_cache_status;
    }
  }
}
```

`src/lib/wards-import.ts`:
```ts
type WardProperties = {
  ward_id: number;
  ward_name: string;
  ward_name_kn: string;
  Corporation: string;
  corporation_kn: string;
  zone: string;
};

export type WardFeature = GeoJSON.Feature<GeoJSON.Polygon, WardProperties>;

export type WardRow = {
  id: number;
  nameEn: string;
  nameKn: string;
  corporation: string;
  corporationKn: string;
  zone: string;
};

export function toWardRow(feature: WardFeature): WardRow {
  const p = feature.properties;
  return {
    id: p.ward_id,
    // ward_name is "25 - Vinayaka Layout" — strip the "NN - " prefix.
    nameEn: p.ward_name.replace(/^\d+\s*-\s*/, ''),
    nameKn: p.ward_name_kn,
    corporation: p.Corporation,
    corporationKn: p.corporation_kn,
    zone: p.zone,
  };
}
```

If `GeoJSON` types are unresolved, `npm install -D @types/geojson`.

- [ ] **Step 6: Write the import script**

`scripts/import-wards.ts`:
```ts
import fs from 'node:fs';
import path from 'node:path';
import { sql } from 'drizzle-orm';
import { db, wards } from '../src/db';
import { toWardRow, type WardFeature } from '../src/lib/wards-import';

const SOURCE = 'data/gba.geojson';
const OUT_DIR = 'public/geo/wards';

async function main() {
  const fc = JSON.parse(fs.readFileSync(SOURCE, 'utf8')) as { features: WardFeature[] };
  if (fc.features.length !== 369) {
    throw new Error(`expected 369 wards, got ${fc.features.length}`);
  }

  fs.mkdirSync(OUT_DIR, { recursive: true });
  for (const f of fc.features) {
    const row = toWardRow(f);
    // Per-ward file for MapLibre: geometry + display props only.
    const slim = {
      type: 'Feature',
      properties: { ward_id: row.id, name_en: row.nameEn, name_kn: row.nameKn },
      geometry: f.geometry,
    };
    fs.writeFileSync(path.join(OUT_DIR, `${row.id}.geojson`), JSON.stringify(slim));
  }

  const rows = fc.features.map(toWardRow);
  await db.insert(wards).values(rows).onConflictDoUpdate({
    target: wards.id,
    set: {
      nameEn: sql`excluded.name_en`,
      nameKn: sql`excluded.name_kn`,
      corporation: sql`excluded.corporation`,
      corporationKn: sql`excluded.corporation_kn`,
      zone: sql`excluded.zone`,
    },
  });
  console.log(`imported ${rows.length} wards, wrote ${rows.length} geojson files`);
  process.exit(0);
}

main().catch((e) => { console.error(e); process.exit(1); });
```

```ts
import en from './en.json';
import kn from './kn.json';

export const locales = ['en', 'kn'] as const;
export type Locale = (typeof locales)[number];

const bundles: Record<Locale, Record<string, string>> = { en, kn };

export function t(locale: Locale, key: string): string {
  return bundles[locale][key] ?? bundles.en[key] ?? key;
}

/** Build a URL for a locale from a canonical (unprefixed) path. */
export function localePath(locale: Locale, path: string): string {
  if (locale === 'en') return path;
  return path === '/' ? '/kn' : `/kn${path}`;
}

/** The same page in the other language, given the current pathname. */
export function alternatePath(currentLocale: Locale, pathname: string): string {
  const canonical = currentLocale === 'kn'
    ? (pathname.replace(/^\/kn/, '') || '/')
    : pathname;
  return localePath(currentLocale === 'en' ? 'kn' : 'en', canonical);
}
```

```ts
import fs from 'node:fs';
import booleanPointInPolygon from '@turf/boolean-point-in-polygon';

type BoundaryFeature = GeoJSON.Feature<GeoJSON.Polygon, { ward_id: number }>;

let features: BoundaryFeature[] | null = null;

export function loadWardBoundaries(path = 'data/gba.geojson'): void {
  const fc = JSON.parse(fs.readFileSync(path, 'utf8')) as { features: BoundaryFeature[] };
  features = fc.features;
}

export function findWardIdByPoint(lng: number, lat: number): number | null {
  if (!features) loadWardBoundaries();
  for (const f of features!) {
    if (booleanPointInPolygon([lng, lat], f)) return f.properties.ward_id;
  }
  return null;
}
```

`src/lib/geocoder.ts`:
```ts
export type GeocodeResult = { lat: number; lng: number };

export interface Geocoder {
  geocode(address: string): Promise<GeocodeResult | null>;
}

// Google Maps ToS (project-dependencies §6.4): geocoding results are used
// server-side only and reduced to a ward. Coordinates must never be sent
// to the browser or stored.
export class GoogleGeocoder implements Geocoder {
  constructor(private readonly apiKey: string) {}

  async geocode(address: string): Promise<GeocodeResult | null> {
    const url = new URL('https://maps.googleapis.com/maps/api/geocode/json');
    url.searchParams.set('address', address);
    url.searchParams.set('region', 'in');
    url.searchParams.set('key', this.apiKey);
    const res = await fetch(url);
    if (!res.ok) throw new Error(`geocoder http ${res.status}`);
    const body = (await res.json()) as {
      status: string;
      results?: { geometry?: { location?: { lat: number; lng: number } } }[];
    };
    const loc = body.results?.[0]?.geometry?.location;
    return loc ?? null;
  }
}
```

`src/lib/ward-lookup.ts`:
```ts
import type { Geocoder } from './geocoder';
import { findWardIdByPoint } from './geo';

export type LookupResult =
  | { ok: true; wardId: number }
  | { ok: false; error: 'not_found' | 'outside_gba' | 'geocode_failed' };

export async function lookupWardId(address: string, geocoder: Geocoder): Promise<LookupResult> {
  let point;
  try {
    point = await geocoder.geocode(address);
  } catch {
    return { ok: false, error: 'geocode_failed' };
  }
  if (!point) return { ok: false, error: 'not_found' };
  const wardId = findWardIdByPoint(point.lng, point.lat);
  return wardId === null ? { ok: false, error: 'outside_gba' } : { ok: true, wardId };
}
```

```ts
import type { APIRoute } from 'astro';
import { eq } from 'drizzle-orm';
import { db, wards } from '../../db';
import { GoogleGeocoder } from '../../lib/geocoder';
import { lookupWardId } from '../../lib/ward-lookup';

const json = (body: object, status = 200) =>
  new Response(JSON.stringify(body), {
    status,
    headers: { 'Content-Type': 'application/json', 'Cache-Control': 'no-store' },
  });

export const POST: APIRoute = async ({ request }) => {
  let address: unknown;
  try {
    ({ address } = await request.json());
  } catch {
    return json({ error: 'bad_request' }, 400);
  }
  if (typeof address !== 'string' || address.trim().length < 5 || address.length > 200) {
    return json({ error: 'bad_request' }, 400);
  }

  const key = process.env.GOOGLE_MAPS_API_KEY;
  if (!key) return json({ error: 'geocode_failed' }, 422);

  const result = await lookupWardId(address.trim(), new GoogleGeocoder(key));
  if (!result.ok) return json({ error: result.error }, 422);

  const [ward] = await db.select().from(wards).where(eq(wards.id, result.wardId));
  if (!ward) return json({ error: 'not_found' }, 422);

  // Ward only — never coordinates (Google Maps ToS, project-dependencies §6.4).
  return json({
    ward: {
      id: ward.id,
      nameEn: ward.nameEn,
      nameKn: ward.nameKn,
      corporation: ward.corporation,
      url: `/ward/${ward.id}`,
      urlKn: `/kn/ward/${ward.id}`,
    },
  });
};
```

```ts
import type { APIRoute } from 'astro';
import { db, wards } from '../db';
import { localePath } from '../i18n';

export const GET: APIRoute = async ({ site }) => {
  const origin = site!.origin;
  const allWards = await db.select({ id: wards.id }).from(wards);
  const canonicalPaths = ['/', ...allWards.map((w) => `/ward/${w.id}`)];

  const urlEntry = (path: string) => {
    const en = origin + localePath('en', path);
    const kn = origin + localePath('kn', path);
    const alternates =
      `<xhtml:link rel="alternate" hreflang="en" href="${en}"/>` +
      `<xhtml:link rel="alternate" hreflang="kn" href="${kn}"/>` +
      `<xhtml:link rel="alternate" hreflang="x-default" href="${en}"/>`;
    return `<url><loc>${en}</loc>${alternates}</url><url><loc>${kn}</loc>${alternates}</url>`;
  };

  const xml =
    '<?xml version="1.0" encoding="UTF-8"?>' +
    '<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9" xmlns:xhtml="http://www.w3.org/1999/xhtml">' +
    canonicalPaths.map(urlEntry).join('') +
    '</urlset>';

  return new Response(xml, {
    headers: { 'Content-Type': 'application/xml', 'Cache-Control': 'public, max-age=0, s-maxage=3600' },
  });
};
```
